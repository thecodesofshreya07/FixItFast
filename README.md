# Online Home Service System

## Description
A web application for booking home services like plumbing, electrical, cleaning, etc.

## Tech Stack
- React (Vite)
- MySQL
- Backend (to be implemented)

## Features
- View services
- Book service
- View bookings
- Payment system

## Run Project
npm install
npm run dev

## Database Setup
1. Run home_service.sql
2. Run advanced_objects.sql (for triggers, procedures, functions)
